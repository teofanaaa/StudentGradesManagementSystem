# StudentGradesManagementSystem
Application manages students' grades. The app can be used by teachers and students as well. 

Requestments:
* JavaMail API 
* Java Activation Framework (JAF)
* Apache Derby

Libraries:
* activation.jar 
* mail.jar
* derby.jar
* derbyclient.jar
* derbyrun.jar
* derbytools.jar
